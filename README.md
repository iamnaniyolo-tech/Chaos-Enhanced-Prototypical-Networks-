# Chaos-Enhanced-Prototypical-Networks-
Chaos-Enhanced Prototypical Networks for Few Shot Medical image classification


Using of deep learning techniques for the diagnosis of neuro-oncological pathologies is heavily restricted by the severe scarcity of properly labelled medical data. Frameworks such as Few-Shot Learning (FSL), particularly metric-based Prototypical Networks, solves this issue by generalizing the class from a minimal samples. When applied on complex brain tumor MRI’s, they suffer from acute “prototype bias, high intra-class variance and inherent morphological noise makes the class centroids to physically shift, causing severe feature entanglement and geometric instability. To overcome this without the increasing the computational overhead and the risk of hallucination frorm the generative models, this work proposes the Chaos-Enhanced Prototypical Network (CE-ProtoNet). By integrating a discrete Logistic Chaos Module (LCM) into a selectively fine-tuned ResNet-18 backbone, creating a new architecture which operates with strict O(1) algorithmic complexity. The LCM injects bounded, deterministic ergodic perturbations directly into support embeddings during episodic training processes. This actively simulates continuous morphological variance, forcing the network to extract noise-invariant biological markers. When evaluated on a 4-way, 5-shot brain tumor classification task, and rigorous ablation studies have shown an optimal regularization threshold at a 15% chaotic perturbation intensity (λ = 0.15). This addition successfully separated visually ambiguous clusters—confirmed by t-SNE geometric projections—improving the baseline accuracy from 79.70% to 84.52% and achieving a peak macro F1-score of 0.8354. The results show CE-ProtoNet as a highly parameter-efficient, mathematically robust framework for high-precision clinical diagnostics in data-scarce environments.
